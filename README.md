# megadecrypter-js

This bookmarklet decrypts `mega://enc2` links on browser without megadonwloader.

## Requirements
- Any browser which can execute Javascript.

## Installation
See [document](https://msglight4874.github.io/megadecrypter-js/).
## Usage

Click the bookmarklet.
